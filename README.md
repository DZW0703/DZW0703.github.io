# DZW0703.github.io — Jekyll 个人主页

这是为 DZW0703 搭建的 Jekyll 个人主页模板。

快速部署
1. 确认仓库 `main` 分支为发布分支（Settings → Pages → Branch: main / Folder: / (root)）。
2. 在仓库根提交本次更改，GitHub Pages 会自动使用内置的 Jekyll 进行构建。
3. 自定义站点信息：编辑 `_config.yml` 中的 `title`、`email`、`description` 与 `url` 项。

文章管理
- 在 `_posts/` 中添加 Markdown 文件，文件名格式为 `YYYY-MM-DD-title.md`。

自定义样式与脚本
- 样式在 `assets/css/style.css`，脚本在 `assets/js/main.js`。
